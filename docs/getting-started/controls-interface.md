# Controls & Interface

Master the controls and interface of Ultima Online to enhance your gameplay experience in New Dawn.

## Basic Controls

### Movement

- **Click & Hold** - Walk in a direction
- **Double Click** - Run/sprint (uses stamina)
- **Arrow Keys** - Alternative movement method
- **NumPad** - Movement control option

### Mouse Controls

- **Single Click** - Select/target
- **Double Click** - Use/activate items, open containers
- **Right Click** - Context menu (client dependent)
- **Drag & Drop** - Move items between containers

## Interface Overview

### Main Game Window

The primary view shows:

- Your character and surroundings
- Other players and NPCs
- Monsters and creatures
- Objects and terrain

### Status Bar

Your health bar displays:

- **Red Bar** - Hit Points (HP)
- **Orange Bar** - Stamina
- **Blue Bar** - Mana

### Backpack

Access your inventory:

- **Open:** Double-click your backpack icon
- **Organization:** Arrange items as you prefer
- **Containers:** Use bags/pouches to organize

### Paperdoll

View your character equipment:

- **Open:** Double-click yourself or press hotkey
- **Equipment Slots:** Wear armor, weapons, jewelry
- **Stats Display:** View current stats

## Essential Commands

### Communication

```
[say] - Normal speech (no prefix needed)
/yell - Yell (larger range)
/whisper - Whisper (smaller range)
/party - Party chat
/guild - Guild chat
/alliance - Alliance chat
```

### General Commands

```
/help - Display help menu
/stuck - Teleport if stuck in geometry
/who - See players online
/time - Display server time
/stats - View character statistics
```

### Social Commands

```
/friend [name] - Add to friends list
/ignore [name] - Ignore a player
/petition - Contact staff (if available)
```

## Targeting System

### Target Types

- **Beneficial** - Healing, buffing (friendly cursor)
- **Harmful** - Attacking, debuffing (hostile cursor)
- **Neutral** - Information, identification

### Target Macros

Set up quick targeting:

- **Target Self** - Quickly heal/buff yourself
- **Target Last** - Attack same target
- **Target Next/Previous** - Cycle through targets

## Hotkeys & Macros

### Default Hotkeys

Common default bindings:

| Key | Action |
|-----|--------|
| Tab | Toggle war/peace mode |
| Alt + Q | Quit game |
| Ctrl + Shift | Highlight objects |
| I | Open backpack |
| C | Open character paperdoll |
| P | Open skills |

### Creating Macros

Use macros to streamline gameplay:

1. **Access Macro Menu** - Through client options
2. **Record Actions** - Create action sequences
3. **Assign Hotkey** - Bind to keyboard
4. **Test & Refine** - Ensure it works correctly

!!! example "Useful Macros"
    - **Bandage Self** - Use bandages on yourself
    - **Drink Heal Potion** - Quick healing
    - **Attack Nearest** - Target closest enemy
    - **Peace Mode** - Quick peace toggle

## Tips for Efficient Play

### Organization

!!! tip "Stay Organized"
    - Use bags to categorize items (potions, resources, gems)
    - Label containers with appropriate names
    - Create a system you can remember
    - Keep bank organized like your backpack

### Hotbar Setup

Recommended hotbar items:

1. **Healing** - Bandages, potions
2. **Weapons** - Quick weapon swap
3. **Spells** - Frequently used spells
4. **Tools** - Common tools (pickaxe, hatchet)
5. **Recall Runes** - Travel locations

### Screen Layout

Optimize your screen layout:

- Position health bars where visible
- Keep backpack accessible
- Hotbar in reach of mouse
- Spell book or skills nearby

## Advanced Controls

## Troubleshooting

### Common Issues

**Game is laggy:**
- Check network connection
- Reduce graphics settings
- Close background applications

**Can't click items:**
- Ensure client is focused
- Check for UI overlaps
- Restart client if needed

**Hotkeys not working:**
- Verify key bindings
- Check for conflicts
- Re-record macro
