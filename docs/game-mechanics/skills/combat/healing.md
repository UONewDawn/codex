# Healing

Use bandages to heal yourself and others.

## Overview

Healing allows you to restore hit points using bandages. This essential survival skill is crucial for all combat builds and provides self-sufficiency in dangerous situations.

## Effects

### Basic Healing

- **Heal Hit Points** - Restore HP over time
- **Cure Poison** - Remove poison
- **Resurrect** - Revive dead players
