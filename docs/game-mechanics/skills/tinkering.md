# Tinkering

Craft tools, traps, and mechanical items.

## Overview

Tinkering allows you to create various tools and mechanical items including lockpicks, tools for other crafting skills, traps, and specialty items. Essential support skill for crafters.

## Effects

### Tool Crafting
Create tools for other skills:
- Lockpicks (for [Lockpicking](lockpicking.md))
- Sewing kits (for [Tailoring](tailoring.md))
- Tinker tools (self-replicating)
- Various specialty tools

### Trap Crafting
Create mechanical traps:
- Trap boxes
- Container traps
- Training traps
- Various trap types

### Specialty Items
- Clocks and watches
- Keys
- Jewelry
- Mechanical items

## Training

### Early (0-30)
- Buy ingots from NPC
- Craft tinker tools
- Make simple items

### Intermediate (30-60)
- Create lockpicks
- Make sewing kits
- Craft various tools

### Advanced (60-85)
- Create all tools
- Make specialty items
- Exceptional quality

### Master (85-100)
- Expert tinkerer
- All recipes mastered
- Exceptional items regularly

## Materials

### Tools
- **Tinker Tools** - Required for tinkering
- Can craft more tinker tools (self-sustaining)

### Resources
- Iron ingots (primary material)
- Gems (for some items)
- Various components

## Economic Value

### Essential Services
- Lockpicks always in demand
- Tools for other crafters
- Constant need
- Steady income

### Profit Sources
- Sell lockpicks to thieves and treasure hunters
- Tool sales to crafters
- Specialty item orders
- Bulk tool production

## Tool Requirements

### Lockpicks
- Essential for [Lockpicking](lockpicking.md)
- Consumed on failure
- High demand item
- Easy to craft in bulk

### Sewing Kits
- Required for [Tailoring](tailoring.md)
- Last many uses
- Moderate demand
- Good profit margins

### Other Tools
- Various crafting tools
- Specialty items
- Custom orders
