# Lumberjacking

Harvest wood from trees and enhance axe damage.

## Overview

Lumberjacking allows you to chop trees for logs and provides a damage bonus when using axe weapons. Essential for crafters who need wood and beneficial for axe warriors.

## Effects

### Resource Gathering

- Chop trees for logs
- Different tree types yield different wood (if available)
- Logs convert to boards for crafting
- Renewable resource

### Combat Bonus

When using axe weapons:

- Damage bonus to axe attacks
- Stacks with other combat skills
- Makes axes more viable
