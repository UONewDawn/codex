# Fishing

Catch fish and treasure from water.

## Overview

Fishing allows you to catch fish from any body of water. Beyond basic fish for food, skilled fishermen can catch rare fish, treasure, and special items.

## Effects

### Fish Catching

- Catch fish for cooking
- Various fish types
- Success rate based on skill
- Automatic weight calculation

### Special Catches

At higher skills:

- Treasure chests
- Special fish
- Rare finds
