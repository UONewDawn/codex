# Stealth

Move while hidden without being revealed.

## Overview

Stealth allows you to move while hidden, staying invisible to others. Essential complement to Hiding skill, enabling mobile stealth gameplay for scouts, thieves, and assassins.

## Effects

### Hidden Movement

- Move while remaining hidden
- Walk speed only (no running)
- Stay invisible to others
- Essential mobility for stealth builds

### Movement Restrictions

- Can only walk (reduced speed)
- Limited number of steps
- Cannot run or mount
- Must already be hidden
- Requires [Hiding](hiding.md) skill

### Breaking Stealth

Stealth breaks when you:

- Run or move too fast
- Attack or cast spells
- Use certain skills
- Take damage

## Training

### Requirements

- Must have [Hiding](hiding.md) skill
- Need 80+ Hiding to begin Stealth training
- Use while hidden

### Active Training

Use Stealth while hidden:

- Hide first
- Then move with Stealth active
- Gains on each step
- Practice anywhere

### Training Tips

!!! warning
    These need refinement. Confirm your template in the Discord community.

- Macro hide and stealth walking
- Circle in safe areas
- Slow but steady gains
- Train to 100 for best results

!!! tip "Hiding Requirement"
    You must have at least 80 Hiding before you can start training Stealth.
